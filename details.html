Gesture Control

The Future is today

Hand gesture recognition system is used for establishing communication between computers and humans. Using a simple gesture we would be able to control specific functionalities of the computer, which helps to avoid the use of mouse . We would like to make a computer based application which identifies live motion gesture recognition using a webcam and produce corresponding results.This project is a combination of live motion detection and gesture identification. This application uses the webcam to detect gesture made by us and perform basic operations accordingly. The user has to perform a particular gesture. The webcam captures this and identifies the gesture, recognizes it and performs the action corresponding to it. This application can be made to run in the background while the user runs other programs and applications. This is very useful for hands-free approach.
